# TASK 3 - Provision Docker Container using Terraform

This task provisions an nginx container locally using Terraform and Docker.

## Steps Done
- Used Docker provider
- Pulled nginx image
- Ran a container on port 8080

## Commands Used
- terraform init
- terraform plan
- terraform apply
- terraform destroy
- terraform state list
